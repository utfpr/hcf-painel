# HCF Painel

## Pré-requisitos

- [NodeJS >= 18.16](https://nodejs.org/en)
- [Yarn](https://yarnpkg.com)

## Configuração

Após a instalação dos programas necessários, faça uma cópia do arquivo `.env.example` e renomeie como `.env`.

## Execução

No terminal, digite os comandos abaixo:

```shell
$ yarn install
$ yarn start
```
